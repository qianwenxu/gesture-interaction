mymusicplay是控制网易云音乐暂停/播放、上一首、下一首、音量升高、音量降低的代码。
runLive.py用于Windows系统的实时手势交互
runLive_linux.py用于Linux（ubuntu）系统的实时手势交互
代码中使用了文件夹"./Saved/"，里面存储了训练好的模型，由于模型太大，所以这里没有上传。生成模型的代码上传了。
parameters.py文件中'./Datasets/osd/Dataset'是数据集，也由于太大，这里没有上传。
